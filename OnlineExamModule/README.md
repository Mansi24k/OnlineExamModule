# OnlineExamModule

OnlineExamModule is an ASP.NET Web Forms based online examination system.

## Project Overview
This project allows users to log in, attempt online exams, and view results.  
It is built using classic ASP.NET Web Forms with C# and SQL Server.

## Features
- User authentication (Login)
- Online exam module
- Test screen with questions
- Result display
- Admin-friendly structure

## Technology Stack
- ASP.NET Web Forms
- C#
- .NET Framework
- SQL Server
- IIS Express

## Project Structure
- `Default.aspx` – Landing page
- `Login.aspx` – User login
- `Home.aspx` – Dashboard
- `Exam.aspx` – Exam page
- `Testscreen.aspx` – Test screen
- `Web.config` – Application configuration

## How to Run the Project
1. Open the solution in Visual Studio
2. Update the database connection string in `Web.config`
3. Restore NuGet packages (if required)
4. Set `Login.aspx` or `Default.aspx` as Start Page
5. Press **F5** to run using IIS Express

## Notes
- Ensure SQL Server is running on your system
- Database must be available before running the project
